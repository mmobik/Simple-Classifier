# Simple-Classifier

Проект представляет собой эволюцию от базовых алгоритмов классификации до полноценной нейросети для распознавания рукописных цифр (0–9). Учебная реализация для глубокого понимания принципов машинного обучения.

## Roadmap (план изучения и реализации)

### Этап 1. Решающее дерево (Decision Tree)
- Реализация с нуля на Python
- Критерий разделения — коэффициент Джини
- Бинарная классификация (0/1)
- Рекурсивное построение с контролем глубины
- Визуализация структуры дерева
- Анализ сложности алгоритма

### Этап 2. Энтропия и информационный выигрыш
- Изучение энтропии как меры неопределенности
- Сравнение с коэффициентом Джини
- Реализация энтропии в решающем дереве
- Анализ влияния выбора критерия на качество

### Этап 3. Алгоритмы классификации
- Логистическая регрессия
- Метод k-ближайших соседей (kNN)
- Сравнение с решающим деревом
- Оценка метрик: точность, полнота, F1-мера

### Этап 4. Распознавание рукописных цифр
- Датасет MNIST (0–9)
- One-hot encoding целевых меток
- Полносвязная нейросеть
- Softmax как функция активации
- Кросс-энтропия как функция потерь
- Обучение и предсказание

### Технологический стек
- Python 3.x
- NumPy — матричные операции
- Pandas — работа с данными
- Matplotlib — визуализация

### Конечная цель
Построение модели, способной распознавать рукописные цифры с картинки с точностью не менее 95% на тестовых данных.
