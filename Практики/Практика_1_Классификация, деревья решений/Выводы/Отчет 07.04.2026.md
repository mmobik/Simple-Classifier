**Сложность DecisionTreeClassifier**
1. Т.к 1 признак, то проход для него O(1)
2. Для каждого порога (уникального значения X) сложность O(m), где m - уникальные значения
3. На каждом уровне дерева проходимся по всем данным для узла n O(n), где n  >= m
4. На одном узле сложность O(n * m)
5. Глубина depth
6. Итоговая сложность: O(n * m * depth), где худший случай: O(n^2 * depth), когда все значения в выборке X уникальны.
7. Сложность из сикит лерн O(n × f × log n × d), где f - колов признаков
